* 알고리즘별 폴더를 나누어 기록할 것입니다.
* 파일명은 문제번호_bj(백준) 형태를 따릅니다.
* 파이썬으로 진행합니다.
-------------------------------------------------------
* 폴더 구조
    * DS1,DS2 (자료구조) - 스택, 큐, 덱, 등
    * String (문자열)
    * Greedy (탐욕법)
    * Implementation(구현)
    * Graph_Traversal(그래프 탐색)
    * Brute_Force (완전 탐색)
    * DP1 (동적 계획법)
    * Binary_search (이분 탐색)
    * Backtracking (백 트래킹)
    * Divide_and_conquer (분할정복)
    * Prefix_sum (누적 합)
    * Shortest_path (최단거리)
    * Two_pointer (투포인터)
    * Math (수학)
    * Tree (트리)
